# WP Career Board

The community-powered job board for WordPress.

**Free core plugin** — works standalone, enhanced with BuddyPress/BuddyBoss, first-class Reign and BuddyX Pro theme support.

- **Slug:** `wp-career-board`
- **Requires:** WP 6.9+, PHP 8.1+
- **Pro version:** [WP Career Board Pro](https://wpcareerboard.com)
