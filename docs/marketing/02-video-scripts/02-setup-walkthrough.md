# Video Script: Installation & Setup Walkthrough

**Duration:** 3:00
**Purpose:** Walk a new user through installing and running the Setup Wizard
**Tone:** Tutorial — calm, clear, step by step
**Target:** WordPress site owner who just downloaded WP Career Board

---

## Script

### INTRO (0:00 – 0:15)

**VISUAL:** WordPress Plugins page

**NARRATOR:**
"In this video I'll show you how to install WP Career Board and get a job board running. The whole thing takes about three minutes."

---

### INSTALL (0:15 – 0:45)

**VISUAL:** Plugins → Add New → Upload Plugin

**NARRATOR:**
"Go to Plugins → Add New → Upload Plugin. Choose the wp-career-board.zip file you downloaded. Click Install Now."

**VISUAL:** Install complete screen → Activate Plugin button

**NARRATOR:**
"Once it's installed, click Activate Plugin."

**VISUAL:** WordPress admin — new WP Career Board menu item in the sidebar

**NARRATOR:**
"You'll see the WP Career Board menu appear in the sidebar."

---

### SETUP WIZARD (0:45 – 1:45)

**VISUAL:** WP Career Board → Settings → Run Setup Wizard button in page header

**NARRATOR:**
"Go to WP Career Board → Settings. You'll see a Setup Wizard button in the top right. Click it."

**VISUAL:** Setup Wizard — Step 1: Welcome screen

**NARRATOR:**
"The wizard has two steps. The first step is a welcome screen — it tells you what it's about to create."

**VISUAL:** Clicking Create Pages & Continue — pages being created

**NARRATOR:**
"Click Create Pages and Continue. The wizard creates four pages — Jobs, Post a Job, Employer Dashboard, and Candidate Dashboard — and inserts the correct block on each one automatically."

**VISUAL:** Step 2 — pages listed with links

**NARRATOR:**
"Step two shows you the pages it created. Click the Jobs link to see your job board."

**VISUAL:** Empty job listings page — "No jobs found"

**NARRATOR:**
"It's empty for now — that's expected. We'll add a job in a moment."

---

### SETTINGS OVERVIEW (1:45 – 2:15)

**VISUAL:** Settings → Job Listings tab

**NARRATOR:**
"Back in WP Career Board → Settings, there are four tabs. Job Listings controls whether jobs auto-publish or go to a moderation queue. The default is manual moderation — you approve each job before it goes live. If you want jobs to publish immediately, toggle Auto-Publish Jobs on."

**VISUAL:** Settings → Notifications tab

**NARRATOR:**
"The Notifications tab is where you configure the admin email address and toggle which notification emails fire."

**VISUAL:** Settings → Pages tab

**NARRATOR:**
"The Pages tab lets you change which pages the plugin uses, if you want to reassign them later."

---

### FIRST JOB (2:15 – 2:45)

**VISUAL:** Frontend Employer Dashboard → Post a Job button

**NARRATOR:**
"Let's post a test job. Open the Employer Dashboard page — you'll be registered automatically as an employer when you access it as an admin. Click Post a Job."

**VISUAL:** Job form — fill in title, location, type, description

**NARRATOR:**
"Fill in a job title, location, type, and a quick description. Click Review, then Submit."

**VISUAL:** Job appears in admin moderation queue OR publishes immediately depending on setting

**NARRATOR:**
"If auto-publish is off, the job lands in the moderation queue — go to WP Career Board → Jobs, find the pending job, and click Approve. If auto-publish is on, it's live immediately."

**VISUAL:** Jobs page — job card visible in listings

**NARRATOR:**
"Now the job appears on your jobs page. Candidates can find it, apply, and you'll see their application in the Employer Dashboard."

---

### CLOSE (2:45 – 3:00)

**VISUAL:** WP Career Board docs page

**NARRATOR:**
"That's the basic setup. For BuddyPress integration, Reign Theme configuration, and more advanced settings, check the documentation at the link below."

**ON-SCREEN TEXT:** wbcomdesigns.com/docs/wp-career-board/

---
