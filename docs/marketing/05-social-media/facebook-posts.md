## Post 1: Launch — WordPress Groups

We just released WP Career Board — a job board plugin for WordPress.

Free version includes everything you'd expect from a paid plugin:
- Employer and candidate dashboards (frontend — no wp-admin)
- Live job search: keyword, location, type, salary, remote filter
- Application tracking with 5 statuses
- Guest applications — candidates apply without registering
- BuddyPress activity stream integration
- reCAPTCHA v3, GDPR, JobPosting schema — all included

No shortcodes. No jQuery. Built on Gutenberg blocks and the WordPress Interactivity API.

Pro version adds: Kanban hiring pipeline, multi-board, resume search, Stripe monetization, AI writing.

Worth checking if you've been putting off adding a job board to your community.

## Post 3: BuddyPress Groups

If you're running a BuddyPress community and want to add a job board, WP Career Board integrates automatically.

When active alongside BuddyPress:
- Job posts appear in the site-wide activity stream
- Employer and candidate roles become BuddyPress member types (so you can filter the member directory)
- Candidate profiles link to their BuddyPress member profile
- Works with BuddyBoss Platform too

Free to download. Pro version available for Kanban pipeline, resume search, and Stripe monetization.

## Post 4: Agency/Freelancer Group

Agency developers: if a client ever asks for a job board on their WordPress site, here's what we'd suggest now.

WP Career Board is block-first — every screen is a Gutenberg block. The architecture uses the Interactivity API, REST API with typed endpoints, and the WordPress Abilities API. No jQuery, no admin-ajax, no shortcodes.

For clients, that means it works with their existing theme and they can manage it from the familiar block editor.

For you, it means the extension hooks are clean and the permissions model is auditable.

Free version is a complete job board. Pro adds pipeline, multi-board, resume search, and Stripe.
