**Thread: What we built**

1/ We just launched WP Career Board — a full job board plugin for WordPress communities.

Free. Block-first. No jQuery. No shortcodes.

2/ For employers: post jobs from a frontend dashboard, review every applicant, update status with one click. No wp-admin access required.

For candidates: apply, track status, save jobs. Apply without an account.

3/ The tech stack:
- Gutenberg blocks (no shortcodes)
- WordPress Interactivity API (no jQuery)
- REST API at /wp-json/wcb/v1/
- WordPress Abilities API (no capability strings)
- schema.org/JobPosting on every job page

4/ Comes with reCAPTCHA v3, GDPR export/erasure, email notifications, and BuddyPress integration — all in the free version.

5/ Included with Reign Theme. Works with BuddyX Pro and BuddyBoss Platform.
