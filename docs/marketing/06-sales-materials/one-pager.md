## THE PROBLEM

Most WordPress job board plugins were built before the block editor. They use shortcodes to place pages, jQuery for every interaction, and custom templates that fight your theme. Basic features like guest applications, spam protection, and GDPR compliance cost extra.

---

## THE SOLUTION

WP Career Board is built entirely on Gutenberg blocks and the WordPress Interactivity API. Every screen is a block. Every interaction is native. Everything you'd expect a modern job board to include is in the free version.

---

## FREE VERSION INCLUDES

**For employers**
- Post and manage jobs from the frontend — no wp-admin access needed
- Receive and review applications with one-click status updates
- Public company profile with logo, description, and location

**For candidates**
- Filterable job directory — live search, salary range, remote toggle
- Application dashboard with real-time status tracking
- Guest applications — apply without creating an account

**For admins**
- Job moderation queue (or auto-publish)
- Customizable email notifications
- reCAPTCHA v3 spam protection (invisible)
- GDPR — personal data export and erasure
- JobPosting schema.org on every job page

**Integrations (automatic)**
- Reign Theme — design system adopted automatically, bundled in Reign license
- BuddyPress / BuddyBoss Platform — activity streams, member types
- BuddyX Pro — design integration
