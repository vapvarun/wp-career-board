# Welcome Email Sequence — WP Career Board

Triggered when a new user downloads / activates WP Career Board (free).
Delivered via the Wbcom account email system.

---

## Email 1: Immediate — Setup

**Subject A:** Your WP Career Board is ready to set up
**Subject B:** One wizard away from a live job board

**Body:**

Hi [First Name],

Thanks for installing WP Career Board.

The fastest way to get your job board live: run the **Setup Wizard**. It creates all the pages you need (Jobs, Post a Job, Employer Dashboard, Candidate Dashboard) and inserts the right blocks automatically.

**Run it now:** Go to WP Career Board → Settings → Run Setup Wizard in your WordPress admin.

The whole thing takes about two minutes.

If you run into anything — pages not showing, emails not sending, filters not working — check the [Troubleshooting guide](https://wbcomdesigns.com/docs/wp-career-board/troubleshooting/) first.

— The Wbcom Team

---

## Email 2: Day 2 — First Job

**Subject A:** Have you posted your first job yet?
**Subject B:** Test your job board in 3 minutes

**Body:**

Hi [First Name],

The quickest way to see your job board working end to end: post a test job.

1. Go to your Employer Dashboard page (created by the wizard)
2. Click **Post a Job**
3. Fill in a title, location, and description
4. Submit

The job will appear on your Jobs page immediately (if auto-publish is on) or land in the moderation queue (if you want to approve jobs first).

Candidates can apply, you can review the application in the Employer Dashboard, and update the status — the whole workflow in a few minutes.

**Two things worth checking while you're there:**
- Go to **WP Career Board → Settings → Notifications** and make sure your admin email is set correctly
- If you're running BuddyPress, check that job activity is appearing in your activity stream

Let us know if you need anything.

— The Wbcom Team

## Email 3: Day 5 — BuddyPress/Reign angle

**Subject A:** Running BuddyPress? Here's what's already connected.
**Subject B:** WP Career Board + Reign — what activates automatically

**Body:**

Hi [First Name],

A quick note on integrations — a few things activate automatically based on what else is running on your site.

**If you have BuddyPress or BuddyBoss Platform:**
- Job posts appear in the site-wide activity stream
- Employer and candidate roles are registered as BuddyPress member types
- No configuration needed — it detected BuddyPress on activation

**If you're using Reign Theme:**
- Job listings, dashboards, and forms inherit Reign's design system
- Control colors from Appearance → Customize → Career Board
- WP Career Board is included in your Reign license — you already have it

**If you're using BuddyX Pro:**
- The design integration activates automatically (same as Reign)

If any of these apply to your site and something doesn't look right, [check the integration docs](https://wbcomdesigns.com/docs/wp-career-board/integrations/).

— The Wbcom Team

## Email 4: Day 10 — Pro upgrade

**Subject A:** What WP Career Board Pro adds (and who needs it)
**Subject B:** When to upgrade to Pro

**Body:**

Hi [First Name],

The free version handles the full job board workflow. Pro is for specific scenarios:

**You need Pro if:**

- You want to track candidates in a Kanban pipeline (Screening → Interview → Offer → Hired) instead of the 5-status list
- You need multiple independent job boards from one install ("Tech Jobs", "Remote Only", etc.)
- You want to charge employers for job postings via Stripe
- Employers need to search your candidate resume database
- You want candidates to set up job alerts for matching listings
- You need custom fields on job listings, company profiles, or candidate profiles

**You don't need Pro if:**
The free version covers everything you need — posting jobs, reviewing applications, running a single board for your community.

[See everything Pro includes →](https://wbcomdesigns.com/downloads/wp-career-board-pro/)

— The Wbcom Team

P.S. If you're on the Reign Theme Developer license, ask about the Pro bundle.
