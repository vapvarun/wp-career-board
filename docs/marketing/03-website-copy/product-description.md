## Standard Description (250 words)

WP Career Board is a full-featured job board plugin built entirely on Gutenberg blocks and the WordPress Interactivity API. No shortcodes. No jQuery. No page reloads.

**What you get in the free version:**

- Filterable job listings with live search — keyword, location, type, category, salary, remote policy
- Employer Dashboard — post jobs, manage listings, and review applicants from the frontend
- Candidate Dashboard — track applications, save jobs, and monitor status at every stage
- Company profiles — public pages with logo, description, team size, and location
- Application tracking — five statuses (Submitted, Reviewing, Shortlisted, Rejected, Hired)
- Guest applications — candidates apply without registering
- Email notifications — automatic emails at every key event
- Job moderation — approve jobs before they go live (or use auto-publish)
- reCAPTCHA v3 — invisible spam protection on job forms and applications
- JobPosting schema.org — every job page outputs valid structured data for Google Jobs
- GDPR — personal data export and erasure via the WP Privacy API
- BuddyPress integration — job activity in feeds, employer/candidate member types
- Reign Theme — included in Reign Theme licenses, integrates automatically

**WP Career Board Pro adds:** Application Pipeline (Kanban), Multi-Board Engine, Resume Builder, Job Alerts, Job Map, WooCommerce Credit System, Custom Field Builder, AI writing assistance, and more.

Built by Wbcom Designs — makers of Reign Theme, BuddyX Pro, and 100+ WordPress plugins.

## Detailed Description (for wbcomdesigns.com product page)

### A job board that belongs in your WordPress site — not bolted on

Most WordPress job board plugins were built years ago, before the block editor existed. They use shortcodes to place pages, jQuery for interactivity, and custom page templates that fight your theme. WP Career Board was designed from the ground up for modern WordPress.

Every screen is a Gutenberg block. Job listings, the application form, employer and candidate dashboards, company profiles — you insert them from the block editor and they inherit your theme's design. The Interactivity API handles live filtering, real-time status updates, and form submissions — without a single line of jQuery.

### What the free version covers

**For employers:**
Post a job in four steps. Add title, location, type, salary range, and description. Optionally choose a deadline and toggle it as featured. Once posted, manage all listings and review every application from a clean frontend dashboard — no wp-admin access required. Company profiles give candidates context about who they're applying to.

**For candidates:**
Browse the job directory with live filters — keyword, location, job type, category, salary range, and whether the role is remote. Apply with one form. Track the status of every application in the Candidate Dashboard. Save jobs for later. Apply without creating an account (guest applications).

**For admins:**
Approve or reject jobs from the moderation queue. Configure which pages are used for what, whether jobs auto-publish, and which email notifications fire. Full GDPR personal data handling built in. reCAPTCHA v3 protects both the job form and the application form.

### Community integration

WP Career Board was built for community sites. If your site runs BuddyPress or BuddyBoss Platform, it connects automatically — job posts appear in activity streams, employer and candidate roles map to BuddyPress member types.

Users of Wbcom Designs' Reign Theme get WP Career Board included in their license. Run the Setup Wizard and the job board is live, styled to match the community.

### Built on WordPress core, not against it

| Layer | Technology |
|---|---|
| Frontend | WordPress Interactivity API |
| Pages | Gutenberg blocks |
| API | REST API (`/wp-json/wcb/v1/`) |
| Permissions | WordPress Abilities API |
| SEO | `schema.org/JobPosting` LD+JSON |
| Privacy | WP Privacy API (GDPR) |
| Spam | reCAPTCHA v3 |

### Pro features

WP Career Board Pro extends the free version with:

**Application Pipeline** — Replace the 5-status system with a fully customizable ATS pipeline. Define your own stages (Screening, Technical Test, Final Interview). Drag candidates between stages on a Kanban board. Terminal stages (Hired/Rejected) trigger automatic emails and update counters.

**Multi-Board Engine** — Run multiple independent job boards from one install. "Tech Jobs", "Marketing Jobs", "Remote Only" — each with its own jobs, employers, pipeline stages, and credit pricing. A Board Switcher block lets visitors tab between boards on one page.

**Resume Builder** — Candidates build structured resumes (experience, education, skills, links) inside WordPress. Employers search the resume database and filter by skills, location, and experience level.

**Job Alerts** — Candidates subscribe to keyword + location combinations and receive email notifications when matching jobs are posted.

**Job Map** — A geographic map view of job listings. Candidates filter by radius and see jobs plotted by location.

**Credit System** — Create credit packages at any price, sell them through WooCommerce with any payment gateway, and require employers to spend credits to post jobs. Recurring packages and per-listing pricing both supported. Also integrates with Paid Memberships Pro and MemberPress for membership-based credit allocation.

**Custom Field Builder** — Add custom fields to job listings, company profiles, and candidate profiles — no code required. Text, textarea, number, select, checkbox, date, URL, and file upload. Set visibility per field (public, logged in, employer only, candidate only).

**AI Settings** — AI-assisted job description writing from within the Post a Job form.

**Job Feed** — RSS and JSON feeds for every board. Aggregate on external sites or import into other systems.
