## SECTION 1: THE USE CASE

A single job board works well when all your listings are the same type. But many communities and job board operators run more than one kind of listing:

- A tech community running "Engineering", "Design", and "Product" boards
- A remote-work site running "Full-Time Remote" and "Contract" boards separately
- A membership community where different member tiers access different boards
- An agency managing job boards for multiple clients from one install

---

## SECTION 2: HOW IT WORKS

**Creating a board:**
Go to **WP Career Board → Boards → Add New Board**. Give it a name, a slug, and an optional description. Save. It's live.

**Assigning employers:**
Employers are assigned to one or more boards from their profile. They can only post jobs to boards they're assigned to, and only see applications for those boards.

**Posting to a board:**
When an employer posts a job and more than one board exists, they see a Board dropdown in the job form. They pick the board — the job is isolated to that board.

**Board Switcher block:**
Add the Board Switcher block above your Job Listings block. Visitors see a tab bar — they click a tab and the listings update to show only that board's jobs. No page reload.

---

## SECTION 3: BOARD ISOLATION

Each board is fully independent:

| Setting | Per-board in Pro |
|---|---|
| Jobs | Yes — jobs belong to one board |
| Employers | Yes — assigned per board |
| Applications | Yes — only visible within a board |
| Pipeline stages | Yes — each board configures its own |
| Credit pricing | Yes — different cost per board |

---

## SECTION 4: DEFAULT BOARD

When you create your first board, it becomes the default. Jobs posted without a board selection (or before boards existed) go to the default.

You can change the default board at any time from **WP Career Board → Boards → Set as Default**.

---

## SECTION 5: WHO IT'S FOR

**Community sites:** Run "Jobs" and "Freelance Gigs" as separate boards. Separate employers and candidates who serve different markets.

**Niche job boards:** A premium board ("Featured Listings" — 2 credits) alongside a standard board (1 credit). Same employers, different pricing.

**Agencies:** Manage multiple client job boards from one WordPress install. Each client is an employer assigned only to their board.
