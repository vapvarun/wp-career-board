## Key Messages (use these everywhere)

1. **Block-first** — Every screen is a Gutenberg block. No shortcodes.
2. **Interactivity API** — Live filtering and updates with no jQuery.
3. **Community-aware** — BuddyPress/BuddyBoss integration built in.
4. **Reign included** — WP Career Board is bundled with Reign Theme.
5. **Free is complete** — The free version handles the full employer-to-hire workflow. Pro goes further.
