## SLIDE 2: The Problem

**Title:** Job boards for WordPress are stuck in 2015

**Body:**
- Shortcodes everywhere — paste them and hope for the best
- jQuery-powered filtering — added to every page load whether used or not
- No BuddyPress awareness — the job board is its own island
- Features you need are add-ons — guest applications, reCAPTCHA, GDPR all cost extra

**Visual:** A shortcode-heavy admin screen (generic)

---

## SLIDE 3: The Solution

**Title:** A job board built for modern WordPress

**Body:** WP Career Board uses:
- Gutenberg blocks (not shortcodes)
- WordPress Interactivity API (not jQuery)
- REST API (not admin-ajax.php)
- WordPress Abilities API (not capability strings)

**Visual:** Block editor with Job Listings block being inserted (block-inserter.png)

## SLIDE 8: Tech Stack

**Title:** Built on WordPress core — not on top of it

| Layer | Technology |
|---|---|
| Frontend | WordPress Interactivity API |
| Pages | Gutenberg blocks |
| API | REST API `/wp-json/wcb/v1/` |
| Permissions | WordPress Abilities API |
| SEO | `schema.org/JobPosting` |
| Privacy | WP Privacy API |
| Spam | reCAPTCHA v3 |

**No jQuery. No React. No shortcodes.**
